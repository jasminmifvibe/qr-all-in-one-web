Help

Camera permission

If scanning with the live camera is not working, make sure camera access is allowed for QR All-in-One in your iPhone settings. Without camera permission, the app cannot scan codes live.

Photo library permission

If you want to scan a QR code from an image or save a generated QR code, photo library access may be required depending on the action. Please allow access in iPhone settings if prompted.

Scanning issues

If the app is having trouble scanning a QR code, try improving lighting, holding the device steady, and making sure the code is fully visible and not blurry. If the code is in a screenshot or photo, try importing a clearer image.

Generated QR save or share issues

If you cannot save or share a generated QR code, confirm that the destination app or photo library permissions are available. Restarting the app can also help if the share sheet does not appear correctly.
